# Frontend

Cliente web construido con React y Vite.

## Carpetas

- `src/app/`: providers globales y piezas de enrutamiento.
- `src/components/`: componentes reutilizables de layout y UI.
- `src/features/`: pantallas y logica agrupada por funcionalidad.
- `src/lib/`: utilidades transversales del cliente.
- `src/styles/`: tema y estilos globales.
- `public/`: assets publicos servidos por Vite.

## Script principal

```bash
npm run dev:frontend
```

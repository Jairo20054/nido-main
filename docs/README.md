# Documentacion NIDO

La documentacion del proyecto fue separada por tipo para que sea mas facil encontrar contexto sin mezclarla con el codigo fuente.

## Carpetas

- `guides/`: guias de inicio, implementacion y personalizacion.
- `reports/`: entregables, resumentes, avances y reportes historicos.
- `checklists/`: listas de verificacion y pruebas manuales.
- `references/`: referencias tecnicas, manifiestos y material de apoyo.
- `todos/`: pendientes y notas de trabajo.

Si vas a retomar el proyecto, empieza por `guides/README.md`.

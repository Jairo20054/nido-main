# Backend

API REST construida con Express y Prisma.

## Carpetas

- `src/app.js`: configuracion base de Express.
- `src/routes.js`: registro de rutas principales.
- `src/modules/`: dominio separado por modulo.
- `src/shared/`: middlewares, helpers y servicios comunes.
- `prisma/`: schema y seed de base de datos.

## Script principal

```bash
npm run dev:backend
```
